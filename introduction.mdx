Integrate Unitalk into your apps with this example:

<CodeGroup tabs="JavaScript,Python">
  ```javascript
  const unitalk = require('unitalk-sdk');

  const client = new unitalk.Client({ apiKey: 'your-api-key' });
  const assistant = await client.assistants.create({
    name: 'Customer Support Bot',
    model: 'gpt-4o',
    instructions: 'Help users with billing inquiries.'
  });
  console.log(assistant.id);
  ```
  ```python
  from unitalk import Client

  client = Client(api_key="your-api-key")
  assistant = client.assistants.create(
      name="Customer Support Bot",
      model="gpt-4o",
      instructions="Help users with billing inquiries."
  )
  print(assistant.id)
  ```
</CodeGroup>
